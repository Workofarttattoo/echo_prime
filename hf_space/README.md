# Hugging Face Space: CorpOfLight / echo_prime_demo

## Description
This Space hosts a demo of **ECH0-PRIME**, an autonomous AGI system. The demo provides a simple chat interface powered by the 70B model via the `dashboard_server` backend. Users can ask the system to run simulations, retrieve research papers, or perform code execution.

## Features
- Interactive chat powered by the 70B Together AI model.
- Access to real‑time tools: QuLabInfinite, ArxivScanner, Python code execution.
- Built with Gradio for a clean web UI.
